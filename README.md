## ⚡ FastProxy (终极代理工具)

FastProxy 是一款基于 Chrome Manifest V3 标准重构的专业级代理管理插件。
v7.0+ 版本带来了全新的 UI 设计、多服务器管理以及极速的规则匹配引擎。

## ✨ 核心特性

### 1. 📡 多服务器管理 (Multi-Server)
* **多配置支持**：不再局限于单一服务器，你可以添加多个代理配置（如：公司、家庭、公共节点）。
* **快速切换**：在插件弹窗 (Popup) 顶部即可一键切换当前使用的服务器。
* **协议支持**：完美支持 SOCKS5 和 HTTP 代理协议。

### 2. 📝 专业级规则管理
* **极速匹配**：采用 `Set` 哈希索引 (Hash Map)，即使有数万条规则也能毫秒级匹配。
* **可视化编辑**：
    * 支持 **搜索** 规则。
    * 支持 **双击域名** 直接修改。
    * 支持 **JSON 导入/导出**，方便备份与迁移。
* **智能分流**：
    * 🚀 **代理列表**：强制走代理的域名。
    * 🛡️ **直连列表**：强制不走代理的域名（白名单）。
    * 🌏 **GFWList**：内置自动更新的 GFWList 规则库。

### 3. ☁️ 强大的云端同步
* **多平台支持**：支持 **GitHub Gist** 和 **WebDAV** (如坚果云、Nextcloud)。
* **自动同步**：支持定时自动上传/下载配置，多设备间无缝漫游。
* **隐私安全**：你的 Token 和密码仅保存在本地浏览器，不会上传到任何第三方服务器。
* **兼容性修复**：v7.3.4 专门修复了坚果云等 WebDAV 服务的 `MKCOL 405` 错误。

### 4. 🎨 现代化的 UI 设计
* **侧边栏布局**：设置页面采用 Sidebar 布局，功能分区清晰。
* **深色模式**：完美支持系统级 Dark Mode，夜晚使用不刺眼。
* **延迟测试**：内置 Google 连接延迟测试功能。

## 📦 安装说明

1.  下载本仓库的最新 Release ZIP 包并解压。
2.  打开 Chrome 浏览器，进入扩展管理页 `chrome://extensions/`。
3.  开启右上角 **开发者模式 (Developer mode)**。
4.  点击 **加载已解压的扩展程序 (Load unpacked)**，选择解压后的文件夹。

## 🛠 使用指南

1.  **初始化**：安装后会自动打开设置页，请先点击“新增服务器”配置你的代理节点。
2.  **切换模式**：点击浏览器右上角插件图标：
    * 🤖 **自动分流**：根据 GFWList + 自定义规则自动判断。
    * 🚀 **全局代理**：所有流量强制走代理。
    * 🛡️ **直接连接**：不使用代理。
3.  **添加规则**：
    * **快捷添加**：在任意网页点击插件图标，选择“加入代理列表”或“加入直连列表”。
    * **批量管理**：进入设置页 -> 规则管理，进行搜索、编辑或导入导出。

## 🔒 隐私政策
* 本插件完全离线运行，所有数据存储在你的浏览器本地 (`chrome.storage.local`)。
* 云同步功能仅在你配置并手动授权后，直接与你指定的网盘/GitHub 通信。
* 我们不收集任何用户数据。
